Case Study

Crypto Inheritance Switch

A crypto inheritance workflow app for configuring beneficiaries, trigger rules, and guided recovery instructions for non-technical family members.

Problem

Many holders postpone inheritance planning because existing options are complex, technical, and difficult for beneficiaries to execute under stress.

What Was Built

  • Route-based product skeleton with core workflow sections
  • Beneficiary management flow with verification state transitions
  • Timeline model that records key workflow events
  • Typed state foundation for setup, triggers, and recovery guides

Current Status

  • Project pushed to GitHub
  • Deployed to Vercel production
  • Ready for next milestone on trigger simulation and local persistence
