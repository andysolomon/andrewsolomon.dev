Case Study

React + Supabase Mini App

A compact production-style build with authentication, CRUD, and deployment.

Problem

Needed a compact, credible portfolio demo that proves practical full-stack execution with user auth, protected data access, and clean deploy workflow.

Build Scope

  • React/Next.js frontend with protected routes
  • Supabase auth and row-level data access
  • CRUD workflow with validation and errors

Tech Stack

React, TypeScript, Supabase, Vercel

Outcome

Project is live and versioned publicly. Feature implementation is actively in progress with execution tracking in-repo.
