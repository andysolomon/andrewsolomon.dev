Case Study

Travel Disruption Rebook Bot

A product-focused travel operations app for flight disruption alerts, rebooking support, and compensation claim workflow tracking.

Problem

Frequent flyers lose time and money during cancellations and major delays. Existing tools either stop at alerts or handle claims too late in the flow.

What Was Built

  • Dashboard for active flights, disruption events, and eligibility state
  • EU261-focused compensation rules and explainable decision output
  • Rebooking option scoring with preference-aware ranking
  • Claim timeline and confidence-band UI for payout expectations
  • Settings and scenario-driven testing harness for deterministic validation

Current Status

  • Lint, build, and tests passing in repository validation
  • Deployed to Vercel production
  • Repository connected to GitHub for ongoing updates
