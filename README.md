# andrewsolomon.dev starter

Minimal static portfolio scaffold focused on Upwork credibility.

## Files
- `index.html` home page
- `styles.css` shared styles
- `projects/*.html` three case study templates

## Local preview
Run from this directory:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
